# SkyTickets

В ходе написания курсовой работы, была разработана программа для автоматизации покупки авиабилетов, она позволяет приобрести билет на самолет без участия обслуживающего персонала. Разработанная программа направлена на повышение удобства пассажиров в приобретении билетов.
В данном разделе рассматриваются: структура программного продукта, бизнес-правила, руководство программиста, руководство пользования программой и тесты, которым была подвержена программа.


**Структура программного продукта.**
Рассмотрим структуру программы курсовой работы. Структура продукта состоит из модулей: авторизации, главной формы, формы со всеми рейсами, личного кабинета, формы выбора места, формы оформления покупки, формы со всеми купленными билетами, формы печати билета, формы статистики.


Для достижения решения поставленной задачи данной курсовой работы был выбран для реализации объектно-ориентированный подход. Язык программирования, на котором велась разработка – это C#. В качестве среды разработки была выбрана программа Visual Studio 2022. Рабочее название программы «SkyTickets».
Тип приложения: Оконное клиент-серверное приложение. Целевая платформа: Windows.


После запуска программы открывается окно авторизации, в этом окне можно войти или зарегистрироваться.

После авторизации пользователя встречает главное окно, на котором он может найти какой-то конкретный рейс, перейти к доступным рейсам, перейти к статистике полетов, перейти к списку своих билетов, перейти в личный кабинет.

Если пользователь перейдет к доступным рейсам он увидит окно, на этом окне пользователь может искать нужный рейс с помощью фильтра или поиска по городу, выбрав рейс и пользователь может купить билет на него.

Если пользователь нажмет купить в окне с рейсами он увидит окно для выбора места в самолете. В этом же окне показано какие места свободна, а какие заняты.

После выбора места пользователь увидит окно покупки билета с выбранным местом. После покупки билет добавится в список всех билетов.
